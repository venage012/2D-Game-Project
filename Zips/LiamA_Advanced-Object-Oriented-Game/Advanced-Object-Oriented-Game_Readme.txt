There are a lot of things to disclose with the sent work.

1. There is currently a bug regarding player rotation and the collision with the walls.  as long as you don't rotate with the player against the wall you will be fine
2. There is no UI or debug menu, all intended text for UI and menu are added to the console to demonstrate what it would've said if there was time to implement.
3. There is only one bullet, so repeated fires will rest the one bullet
4. Bullet movement is best with horizontal or vertical shots, any diagonal shots will be aimed at a corner not dynamic with the mouse.

The controls are the arrow keys, with the mouse as the aim function.  Left click with the mouse to fire the bullet.
The goal is to shoot the boxes (in blue) and then fire at the target (in red).  Firing at the target too early will cause the level to reset.

Debug controls are activated by pressing the '~`' key.
Debug controls:
- '1','2','3','T' keys change the level to Level1, Level2, Level3, LevelTest respectively
- '_-' key will decrease the player's speed
- '+=' key will increase the player's speed
